1️⃣ Project Title
# Google Ads Campaign Performance Analytics Pipeline
2️⃣ Short Description
A complete analytics pipeline for Google Ads campaigns: from raw data processing to database storage and interactive Power BI dashboards.

3️⃣ Features / Highlights
## Features
- Clean and transform raw Google Ads export files using Python and pandas
- Store campaign data in MySQL database
- Analyze campaign performance with KPIs like clicks, impressions, CTR, and cost
- Build interactive dashboards in Power BI for visual insights
- 
4️⃣ Technologies / Tools Used
  ## Technologies
- Python (pandas, numpy)
- MySQL
- Power BI
- Excel for raw data exports
  
5️⃣ Project Structure
- `Data_Analysis_Project/Adword-Data-Analysis/`: Core analysis pipeline
    - `python_scripts/`: Scripts for data cleaning and processing
    - `mysql_schema/`: SQL scripts to create database tables
    - `power_bi_dashboards/`: Sample dashboards for campaign analytics

6️⃣ How to Run / Usage
## Usage
1. Export raw campaign data from Google Ads into Excel.
2. Run Python scripts to clean and transform the data.
3. Import cleaned data into MySQL database.
4. Open Power BI dashboards and connect to MySQL to visualize metrics.
